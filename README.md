# dotfiles --- devenv provisioner
## Provision scripts
- Zsh
- Emacs lisp

## Installation
```
sh -c "$(curl -fsSL https://raw.github.com/nabinno/dotfiles/master/install.sh)"
```

## Provision
### Zsh
Launch zsh or run zsh resource.
```
exec zsh -l"
```

### Emacs
Launch Emacs.
```
emacs
```
